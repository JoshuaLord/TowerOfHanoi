Detailed Description

The Box class holds position and color data for a multicolored rectangle. It also has facilities for loading the data to the graphics card and invoking draw commands on the data.

void Box::LoadDataToGraphicsCard ( )
private

Loads the box data to the graphics card.

Creates an array of vertex positions and colors from the data stored in the object, creates an index array, allocates graphics memory for the arrays and loads the arrays into graphics memory. Finally, it sets up pointers to the vertex and color data in the two arrays.
