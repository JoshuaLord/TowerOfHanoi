Detailed Description

The Polygon class holds position and color data for a single colored polygon. It also has facilities for loading the data to the graphics card and invoking draw commands on the data.

void Polygon::LoadDataToGraphicsCard ( )
private

Loads the box data to the graphics card.

Creates an array of vertex positions and colors from the data stored in the object, creates an index array, allocates graphics memory for the arrays and loads the arrays into graphics memory. Finally, it sets up pointers to the vertex and color data positions in the array.
